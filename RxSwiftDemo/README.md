## 動作確認環境

- OS X EI Capitan 10.11.3
- Xcode Version 7.2.1 (7C1002)
- rbenv 1.0.0
- ruby 2.3.0p0 (2015-12-25 revision 53290) [x86_64-darwin15]
- CocoaPods -v 1.0.0.beta.4

## 注意事項

CocoaPodsのバージョンが0.39だとエラーが発生してインストール出来ないので、1.0系（現時点ではまだベータ）を使用します

```
cocoapods -v 1.0.0.beta.4
```
