# ParseLoginDemo

Parse.comが提供している独自ログインを行うデモアプリ

## 概要[WIP]

アプリ起動時にログイン画面を表示し、メールアドレスとパスワードを入力してログインボタンをタップしたら、ログインを行い、ログインの結果を表示する。

## 動作確認環境
- OS X EI Capitan 10.11.2 (15C50)
- Xcode Version 7.2 (7C68)
- iPhone 6s Plus
